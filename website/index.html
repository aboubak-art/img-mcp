Give your AI agents the power to generate images

img-mcp is an open-source MCP server that connects agents to Google's Nano Banana image-generation models through a simple, user-provided API key.

Get started View on GitHub

Features

🎨 Text-to-image

Generate images from natural-language prompts with aspect-ratio and size controls.

🚀 Nano Banana ready

Built for Google's Gemini Flash Image models via the Gemini API.

🔑 Bring your own key

No API key sharing. Users provide their own GOOGLE_API_KEY.

⚡ Zero-install usage

Run instantly with npx -y img-mcp.

🛠️ MCP native

Works with Claude Desktop, Cursor, Windsurf, and any MCP-compatible host.

📦 Open source

MIT licensed. Contributions welcome on GitHub.

Installation

Get an API key from Google AI Studio, then run:

Terminal 
export GOOGLE_API_KEY="your-google-api-key"
npx -y img-mcp

Usage with MCP hosts

Add the server to your MCP host configuration:

claude_desktop_config.json 
{
  "mcpServers": {
    "img-mcp": {
      "command": "npx",
      "args": ["-y", "img-mcp"],
      "env": {
        "GOOGLE_API_KEY": "your-google-api-key"
      }
    }
  }
}

Then ask your agent to generate an image:

Example prompt 
Generate an image of a futuristic city at sunset, 16:9 aspect ratio, 2K resolution.

Tool reference

Argument Type Required Default Description
prompt string Yes — Text description of the image
model string No gemini-3.1-flash-image Nano Banana model ID
n number No 1 Images to generate (1–4)
aspect_ratio string No 1:1 Aspect ratio
image_size string No 1K Image size
output_path string No — Save the first image to disk
