Features

Everything you need to generate images from prompts

From simple text-to-image to advanced reference-based generation and transparent backgrounds.

Text-to-image

Generate high-quality images from natural language prompts with full control over aspect ratio and resolution.

Image-to-image

Pass reference images as base64 or data URIs to guide style, composition, and subject in new generations.

Transparent backgrounds

Generate assets with clean transparent backgrounds, perfect for logos, product shots, and overlays.

Flexible sizes & ratios

Choose from 1:1, 16:9, 9:16, 4:3, 21:9, and more. Render from 512px up to 4K.

Bring your own key

Your Google API key stays in your environment. No shared credentials, no proxy servers.

Zero-install usage

Run instantly with npx -y img-mcp. No global install required.

How it works

From prompt to pixels in three steps

img-mcp translates your agent's requests into image generation calls and returns the results.

1

Set your API key

Grab a free API key from Google AI Studio and export it as GOOGLE_API_KEY.

2

Add to your MCP host

Paste the one-line MCP server config into Claude, Cursor, Windsurf, Kimi, or any compatible host.

3

Ask for images

Tell your agent what to create. img-mcp generates the image and returns it as base64 or saves it to disk.

Quick start

Install and run in seconds

No configuration files, no Docker containers. Just your API key and one command.

terminal 
# 1. Set your Google API key
$ export GOOGLE_API_KEY="your-google-api-key"

# 2. Run the MCP server with npx
$ npx -y img-mcp
